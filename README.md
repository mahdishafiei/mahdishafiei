### About

Computational biologist at [Scripps Research](https://ward.scripps.edu) (Ward Lab / [Briney Lab](https://github.com/brineylab)), developing machine learning methods for antibody engineering and protein design. My work spans language model pre-training at scale, structure-based sequence design, and statistical modeling in healthcare.

**Research interests:** antibody language models, protein design, scaling laws, deep learning for structural biology
